title: 'Creating HCA Reports'
description: How to create Work Unit and HCA reports in BIRT Studio.
---

!!! warning

    The Healthcare Analytics entities and columns are available only from the API and cannot be used in Dataviews or to create reports at this time. To get the Healthcare Analytics data, use the Retrieve Data (POST /v1/commons/data/multi_read) API operation. Retrieve only the WORKUNIT and WORKUNIT_TIMESERIES data and ignore the Employee and Location data.

Requests to create HCA reports are not supported in BIRT Studio. See [this](https://sso-hlp01.gss.mykronos.com/help/datadictionary/_MadcapConversionToProduction/ukg_wfm_production/content/DataDictionaryHelp/DD_entities/Healthcare_Analytics/healthcare_analytics_data_entities_22330.html#healthcare_analytics_data_entities_22330) for more information.
