# Pro WFM Reporting

BIRT Studio is the web-based report designer built into UKG Pro Workforce Management. It lets you build custom reports by dragging and dropping fields onto a canvas — no database knowledge or scripting required.

Use the sidebar to navigate the documentation, or start with the guides below.
